Welcome to LVSG Utils Page

In many java projects its necessary implement same utilities classes. This project is a collection of utilities classes.

MapUtils

In java to create a hashmap with values it's necessary to do:

Map<Object,Object> map = new HashMap<Object,Object>();
map.put("item1",o);
map.put("item2",obj2);

with MapUtils

Map<Object,Object> map = MapUtils.createMap("item1",o,"item2",obj2);
